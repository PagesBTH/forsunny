Web Usability

Choose 10 web usability rules that are considered and implemented for this portfolio website

These are my implementations to make a good website:

  • Use fonts that are easy to read
    • - Arial was a font that I've used for this page, as it's easy on the eyes.

  • Use Proper Headings
    • - Using different colours could help identify context, and importance. IN this case, I wanted the headings to pop out

  • Choose fonts that have good kerning
    • - Using text that has good kerning will make it easier for the user to read text

  • Choose a font family that will tie in with your website theme
    • - This will ensure that Consistency and Patterns are met

  • Proper use of Bold & Italics
    • - If you want emphasis on your words, or to specifically highlight a word/text, italics and bolding will help
    - If you italicize/bold every word, the user will not pay attention to an important part that you're trying to convey.

  • Consistency with the web page
    • - Your website should be consistent with design patterns everywhere
    - This makes it easier for the user to browse/read your website

  • Backgrounds
    • - Choose simple colours for your background, so that navigation, and text reading will be easier

  • Contrasting Colours for text
    • - If you want to make text easy to read, choose colours that contrast with your background colours.

  • Use simple language
    • - Using simple language to describe what you're talking about will accommodate everyone reading the page.

  • Use proper margins and patterns
    • - This is needed if you want to keep your web page consistent, along with making navigation easier and efficient with the user
