    Questions:
  1. The reason of choosing the file format when creating your audio
    2.
    - File 1 was using MP3. This is the most widely used file format for music. We all know how it sounds, so I chose this as the default
    - File 2 was under OGG/Vorbis. I call this a better version of MP3. It's a lossless compression, and can potentially give a better sound experience with the same file size as MP3.
    - File 3 was a WAV format. I knew about this since Microsoft released it for Windows 2000. I wanted to show how large a compressed file is.

  2. The use of audio on your website regarding usability.
    3.
    - If you want the user to clearly hear your audio, use a lossless compression algorithm like OGG.
    - Make sure the microphone you use has a pop-filter, if you're a voice actor. Otherwise, just make sure everything is ok.
    - If you wish to make your website accessible, use mono sound along with lossless/uncompressed audio.

  3. Any thoughts on different audio file formats, e.g., cons, pros, when to use.
    4.
    Type of File Pros Cons When to Use
    MP3 Reduced file size, most compatible and used file formats Potentially poor audio quality Web pages, and music players
    OGG Better sound quality than MP3 with the same sample rate, and or file size Compression still loses some details of the sound file same as MP3
    WAV Can easily be converted into other file types, as it keeps a really high audio sample rate Uncompressed files are gigantic when it comes to file sizes, incredibly inefficient when transmitting data from a server. Podcasts, Mastering Videos and Movies

  4. Description of your editing on the file
    5.
    - File 2 was edited, and converted into the OGG/Vorbis format.
    - Added the Vocode Attribute in Audacity.
    - Reduced the decibel count by -5dB.
